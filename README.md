## Installation
Clone project 
```
  git clone https://github.com/Brayanro/prueba-frontend.git
```

Install dependencies
```
  yarn install or npm install
```

Run proyect in development
```
  yarn dev or npm run dev
```

Run JSON Server
```
  yarn dev or npm run server
```
